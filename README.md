# V2T
**Version 0.0.2**

Voice to text converter app. Still in work.
---
# Usage

**Files**
1. Open any .wav or .flac file

**App**
Ctrl+S - save file
Ctrl+O - open file

Duration is number of amount of seconds that app will listen for audio file
---
# Requirements

1. Python 3
2. Pip
3. Speech_recognition (pip/pip3 install speech_recognition)
